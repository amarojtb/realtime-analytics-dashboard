# Real-Time Analytics Dashboard

A live analytics dashboard with animated Canvas charts, auto-updating KPIs, and smooth real-time data transitions. Built entirely from scratch with zero dependencies.

![Dashboard Preview](https://img.shields.io/badge/demo-live-brightgreen)

## Features

- 📊 **Animated Canvas Charts** — Line, bar, donut charts with smooth transitions
- 📈 **Real-Time Data** — Auto-updating KPIs and sparklines
- 🎨 **Light/Dark Theme** — Toggle with localStorage persistence
- 📱 **Responsive Design** — Works on all screen sizes
- ⚡ **Zero Dependencies** — Pure HTML, CSS, and JavaScript

## Quick Start

1. Clone this repository
2. Open `index.html` in your browser
3. That's it! No build tools or server needed.
